### 10진수를 2진수로 표현하기
>1번째 방법

148 

148 / 2 = 74 + 0
74 / 2 = 37 + 0
37 / 2 = 18 + 1
18 / 2 = 9 + 0
9 / 2 = 4 + 1
4 / 2 = 2 + 0
2 / 2 = 1 + 0
1 / 2 = 0 + 1
= 1001 0100
